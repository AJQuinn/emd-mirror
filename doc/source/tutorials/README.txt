EMD Tutorials
=============

Guides to help get you started with data analysis using the EMD toolbox.

You can download all tutorials as :download:`Python files<../emd_tutorials/emd_tutorials_python.zip>` or :download:`Jupyter notebooks<../emd_tutorials/emd_tutorials_jupyter.zip>`. Individual tutorials can be downloaded from their respective pages.

An Anaconda environment for installing EMD alongside Spyder, Jupyter and iPython can be downloaded :download:`here<https://gitlab.com/emd-dev/emd/-/blob/master/envs/emd_tutorial_conda_env.yml>`
